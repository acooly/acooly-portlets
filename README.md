门户前端组件库
=====

## 简介

门户前端组件库，主要是支持门户前端的一些基础组件，包括：评论、点赞、举报、分享、收藏等。一般提供后端服务接口，OpenAPI，和BOSS管理等功能集成。

有前端网站或portal的项目中，可选使用该组件库的组件快速搭建后端支持功能
